# canreplay  - a tool to replay CAN traffic based on CSV file recording

## Usage

```shell
usage: can_replay_v2.py [-h] --dbcfile DBCFILE [--obdconfig OBDCONFIG] [-i INTERFACE] [--verbose]
                        [--delimiter DELIMITER] [-s SLEEP_INTERVAL_MS] [--dryrun]
                        inputfilename
```
